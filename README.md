# Fake News Detection System

## Project Overview
This project is a Machine Learning based Fake News Detection system that classifies news articles as **Real** or **Fake** using Natural Language Processing techniques.

The model is trained using a dataset containing real and fake news articles and is deployed through a **Streamlit web application** for easy user interaction.

---

## Features

- Detect fake or real news from text input
- Batch news detection using CSV file upload
- Confidence score for predictions
- Interactive web interface using Streamlit

---

## Technologies Used

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression
- Streamlit

---

## Dataset

The dataset consists of two sources:

- Fake News Dataset
- Real News Dataset (Reuters)

These datasets were combined and processed to create a training dataset with **44,000+ news articles**.

---

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Text Vectorization using TF-IDF
4. Model Training using Logistic Regression
5. Model Serialization using Pickle
6. Web Application Development using Streamlit
